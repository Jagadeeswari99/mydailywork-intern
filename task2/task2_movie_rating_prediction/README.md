# Task 2 - Movie Rating Prediction with Python

Built a machine learning model to predict movie ratings using Python and Scikit-learn.

My second task for the My Daily Work Data Science Internship is done — and this one was a personal favourite! ⭐⭐⭐⭐
 
The goal: predict movie ratings using machine learning regression models.
 
What I worked on:
🎞️ Feature engineering from movie metadata (genre, cast, director, runtime, votes)
📊 Exploratory analysis to uncover what actually drives ratings
🤖 Trained & compared multiple regression models:
  → Linear Regression (baseline)
  → Random Forest Regressor (best performance)
📉 Evaluated using MAE, RMSE & R² score
🔍 Identified the most influential features using feature importance
 
Key insight: Viewer vote count & genre combinations were stronger rating predictors than individual cast names alone — which was genuinely surprising!
 
Building this made me appreciate how much domain knowledge (in this case, film & audience psychology) shapes what features you engineer.
